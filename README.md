# 🌸 泰语单词花园

> 一款专为泰语学习者设计的交互式闪卡学习工具，支持卡片背诵、分类浏览、学习进度追踪及泰语真人发音。


## ✨ 功能特点

### 📝 闪卡背诵模式
- **先思考，后验证**：页面默认只显示泰语单词，主动回忆中文释义
- **点击卡片或「显示释义」按钮**：显示中文释义 + 例句
- **发音支持**：点击 🔊 按钮，浏览器自带 TTS 语音播报泰语（Web Speech API）
- **键盘快捷键**：← / → 切换单词
- **自动标记已学**：查看释义后自动标记为已掌握

### 📂 分类浏览
- 按主题分类汇总泰语单词，方便场景化学习
- 点击分类展开查看该分类下所有单词
- 支持自定义分类标签（管理员功能）

### 📊 学习进度追踪
- **总单词数 / 已学数量 / 待学数量** 一目了然
- **分类进度**：每个分类的掌握百分比
- 数据自动保存在浏览器本地（localStorage），**刷新页面、关闭浏览器、重启电脑都不会丢失**

### 🔧 管理员功能（密码保护）
- **添加/编辑/删除单词**：支持泰语、中文、音标、例句、分类标签
- **批量导入**：一次性导入多个单词（格式：泰语, 中文, 音标, 例句, 分类）
- **导入/导出 JSON**：备份或迁移词库数据
- **清空词库**：一键重置所有数据


## 🚀 在线体验

👉 [点击开始背单词](https://dearm122277-web.github.io/word-garden/)


## 📂 词库数据格式

### 批量导入格式（CSV风格）
每行一个单词，用英文逗号分隔：
```

泰语, 中文, 音标, 例句, 分类
สวัสดี, 你好, /sà-wàt-dii/, สวัสดีครับ, 基础
ขอบคุณ, 谢谢, /khàawp-khun/, ขอบคุณมาก, 基础

```

### JSON 格式（用于导入/导出）
```json
[
  {
    "word": "สวัสดี",
    "phonetic": "/sà-wàt-dii/",
    "meaning": "你好",
    "example": "สวัสดีครับ",
    "tags": ["基础", "问候"],
    "difficult": false,
    "learned": false
  }
]
```

❓ 常见问题

Q：数据会丢失吗？
A：不会。所有学习进度和词库数据存储在浏览器本地（localStorage），关闭浏览器或重启电脑都不会重置。

Q：发音功能怎么用？
A：点击单词卡片上的 🔊 按钮，浏览器会自动朗读泰语单词（需保持网络畅通，TTS 依赖在线语音库）。推荐使用 Chrome 浏览器。

Q：怎么进入管理员模式？
A：点击页面底部的「⚙️ 管理员」，输入密码即可。密码请联系作者获取。

Q：怎么批量添加单词？
A：进入管理员模式 → 点击「📥 批量导入」→ 按格式粘贴单词列表 → 一键导入。
